# reearth-plugin-layers-tree
A plugin with a widget of layers tree

## Features:
- Select layer
- Hide/Show layer
- Minimize
- Fold/Unfold folders
## Notice:
- Hidden layers (set from earth editor) will not be listed.
- Expect using with published page and NO auto layers update.
